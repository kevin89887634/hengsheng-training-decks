07 / 22

二选一提问法

避免直接问「你要多大保额」——客户不知道怎么回答,或会报低。

❌ 直接问
  • 你要多大保额？
    → 客户不知道怎么回答
  • 你能放多少钱？
    → 客户怕你多赚,通常报低
✅ 二选一
  • 100万的合适呢,还是75万的合适？
    → 挖掘真实预算
  • 一个月放500块合适呢,还是800块？
    → 客户自己告诉你预算
展现专业度的提问
"你希望我按需求来,还是按预算来跟你计算？"

五大经典异议与应对

❓ 考虑考虑
直接问:你考虑的是第一种还是第二种情况？
→ 筛出真假需求
❓ 没钱再付了
分情况:短期付不出 / 生病理赔 / 取现金值或降保额
→ 展示灵活性
❓ 保险公司破产
资产 vs 负债 + 不会挤兑 + State Guarantee + 再保险
→ 讲机制
❓ 真的会赔吗
拿出理赔数据手册(经理手上有)
→ 用数据说话
❓ Google 不到
说排名是买的,教用 NAIC wink 做专业对比
→ 专业工具
