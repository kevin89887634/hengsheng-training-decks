翻页重播验证

第二页 · 应当从头播

从上一页翻过来,这个动画会从第 0 帧重新播一次(loop=true 时持续循环)。所有 deck 通用。

接入方法

3 步把 LottieFiles 上的动画接进来

  1. 从 LottieFiles 下载 JSON,扔进 assets/lottie/ 目录
  2. 在 deck 的对应 slide 里加一个容器:
<div class="hs-lottie" data-lottie="../../assets/lottie/your-file.json" data-loop="true"></div>

参数:data-loop(默认 false,一次播完停在末帧)· data-speed(默认 1)· 每次翻到这一页都从头播,不用写代码。
