模块 2 · SOP
06 / 12

会中配合 — A 主导 + B/C 同侧

座位定调性,B 退后一步观察全局

❌ 错误布局
  • • A B C 直线 — 像审讯席,C 有压迫感
  • • B 抢话 — 打断 A 专业输出
  • • B 玩手机 — 信任崩塌
✓ 正确布局
  • • A 主导位 — 体现权威
  • • B 与 C 同侧 — 像朋友,无压迫
  • • B 记录 / 观察 — 隐性补位
📝 记录需求
客户原话 + 关键数字 + 反对意见 — A 后续跟进的依据
👀 观察情绪
C 表情/动作 — A 输出过快/不足时,用眼神给 A 信号
🤝 引导介入
"关于这个细节,我觉得 [A] 的见解对您可能更有启发"
🎯 一句话原则
A 负责攻坚(逻辑),B 负责守城(情感 + 细节)。分工确保不因专业而冰冷,不因亲近而失严肃。
模块 2 · SOP
07 / 12

会后复盘 + 泼冷水的艺术

闭环管理 + 风险对冲,把单次成功变长期客户

📋 会后 24h 复盘
  • 成交点 — 哪句话打动了 C?
  • 未达标项 — 哪些顾虑未解决?
  • A 的反馈 — A 觉得哪些环节有改进空间?
  • 下次切入点 — 下次见 C 聊什么?
目的:把单次成功转化为标准化经验,下次更精准
💧 泼冷水 — 风险控制
  • 何时用:成交关键期 / C 冲动时
  • 谁来做:B 做情感平衡 + A 做风险提示
  • 怎么泼:引入客观、略带批判性的视角
  • 为什么不否定:防止"被销售"的抵触心理
本质:"泼冷水" = 风险对冲,不是否定方案
💬 A 风险提示话术模板(成交前必说)
"基于目前的市场波动,我必须从专业角度提醒您一个潜在风险……这个方案不是完美的,但在我们能提供的方案里,这是最适合您当下情况的。"
🎯 一句话原则
复盘是为了 下次更准;泼冷水是为了 客户不后悔。两者合一 = 长期口碑。

场景 1: IUL 退休规划 — 借力 CFT 经理

客户:王总 45 岁,科技公司高管,想用保险做退休金补充

📍 会前(你做什么)
  • 情报收集:年收入 30W+ / 已有 401(k) / 2 个孩子 / 关注退休生活质量
  • 匹配 A:CFT 经理张姐(NLG IUL 讲师 + 退休规划专长)
  • 场地:CBD 安静咖啡厅包间,白天
  • 资料同步:提前 24h 把王总财务画像发给张姐
  • 推崇话术:"张姐专门做高净值客户的退休规划"
📍 会中(你的位置)
  • 座位:张姐正对王总,你坐王总旁边(同侧)
  • 引入:介绍完王总需求后,自然过渡到"张姐"
  • 记录:王总对 S&P 500 指数 / 退休金缺口 / 税收延展 的反馈
  • 情绪观察:王总皱眉时,递水 + 拍肩("张姐您详细说说")
  • 引导深入:技术细节自然转给张姐
✅ 场景后结果(实战验证)
张姐专业输出 IUL Living Life Defender 1500 万保额方案 → 王总当场签意向 → 你后续 1 周跟进签单。
关键转化点:张姐讲"退休金 30 年复利 + Living Benefits 退休后免费领"时,王总眼睛亮了 — 你观察到后顺势推:"张姐把这个给您详细算一下"。
💡 场景要点
高净值客户对专业深度要求高,新人很难 hold 住。借力 CFT 经理 = 让 R 层级帮你背书,你只需做好 "B 的桥梁" 角色。

场景 2: 老客户加保 — 借力产品专家

客户:李姐 52 岁,3 年前买了 Term,今年想加保 + 转 IUL

📍 难点(为什么要借力)
  • 李姐 3 年前买的是 NLG Term 500K,你熟
  • 但她现在想 加保 200K Term + 转 IUL 300K
  • 涉及 保单转换 / 健康告知重审 / 财务核保
  • 你的 IUL 知识不深,讲不清 Max Funding / MEC / Loan
  • 李姐已经信任你,但要她听你讲 IUL — 她会犹豫
📍 解法(借力谁)
  • 匹配 A:恒晟产品专家 Tony(IUL 产品线讲师)
  • 约法:三方会议,Tony 用 20 分钟讲 IUL
  • 你的角色:"我先帮您把 Tony 老师约好,他对这块最专业"
  • 推崇:"李姐,Tony 老师是恒晟 IUL 产品的首席顾问,我自己也是听他讲课才明白的"
  • 情感承接:Tony 讲完后,你接"李姐,这个对您孙子教育金正好"
✅ 场景后结果
Tony 现场演示 IUL 跑分 + Living Benefits 触发条件 → 李姐说"我就找你买" → Tony 推荐回给你签单。
关键转化点:Tony 讲到 "Critical Illness 提前给付" 时,李姐想起自己母亲癌症经历 — 你观察到后,接话:"李姐阿姨身体现在怎么样?"(情感共鸣)
加保 200K Term + IUL 300K = 首年保费 $8,400,佣金 $5,800+
💡 场景要点
老客户加保,信任已经在你身上,但 新产品需要新背书。借力不是"自己不专业",而是"给客户最专业的体验"。

场景 3: 转介绍成交 — 借力 A 老师

客户:陈总 50 岁,通过老客户张哥介绍,素未谋面

📍 难点(为什么必须借力)
  • 陈总 不认识你,只听张哥说过"恒晟有个小兄弟"
  • 见面 = 第一次建立关系,失败成本极高
  • 陈总是企业主,时间值钱,不愿听"小兄弟"慢慢讲
  • 如果单独见,你讲 1 小时可能就被赶走
  • 借力 A 老师(资深讲师) = 把"拜访"升级成"听课"
📍 解法(借力怎么做)
  • 约法:"陈总,刚好我们恒晟的王老师本周在洛杉矶,专做企业主资产传承,张哥也认识他"
  • 匹配 A:王老师(20 年经验,恒晟首席讲师)
  • 场地:陈总公司附近的私房菜包间(他熟悉)
  • 推崇话术:"王老师刚从纽约过来,陈总您难得有空"
  • 你的角色:做东道主,负责介绍 + 倒茶 + 后续跟进
✅ 场景后结果(转介绍黄金案例)
王老师用 30 分钟讲"企业主资产传承 3 大雷区" → 陈总主动问"那我该怎么处理?" → 王老师给方案 → 陈总当场签。
关键转化点:陈总说"你直接给我算算,我现在就决定" — 你顺势递合同。
成交:IUL 500 万 + Annuity 100 万 = 首年保费 $36,000,佣金 $25,000+
⭐ 关键认知
转介绍客户 = 高意向但零信任。你一个人搞不定,但 A 老师 + 半小时 就能破冰。这就是 ABC 法则的威力 — 把"陌生拜访"变成"专业咨询"。

风险与禁忌 — 4 大死亡陷阱

做错一次,客户就再也不信你

❌ 角色错位

表现:B 在现场抢戏 / A 过于强势让 C 受压迫

后果:信任崩塌,客户感到"被销售"

正解:A 主导输出,B 退后观察;B 必要时引导,绝不抢话

❌ 情报断层

表现:B 未向 A 同步关键信息

后果:方案与需求南辕北辙,资源浪费

正解:会前 24h 必发资料;复杂客户加 1 次电话 sync

❌ 推崇过度

表现:"我们 A 是全国第一 / 处理过 XX 总统"

后果:虚假吹捧,专业度现场"破产"

正解:"A 处理过类似案例" — 具体不夸张

❌ 缺乏闭环

表现:会后不复盘,同样错误重复发生

后果:效率停滞,新人成长慢

正解:24h 内做书面复盘 + 团队周会分享

✅ 会前 5 项自检清单(打印贴在桌上)
  • □ C 的 3 大需求 / 痛点 是否清楚?
  • □ A 的专长 是否与 C 需求匹配?
  • □ 推崇话术 是否具体不夸张?
  • □ 座位布局 是否 A 主导 + B/C 同侧?
  • □ 复盘机制 是否提前规划好?
